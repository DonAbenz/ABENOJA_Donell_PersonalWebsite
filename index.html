PORTFOLIO

Point of Sale System

As a final project for my Fundamentals of Database Management System course, my partner Odan, Teves and I develop a Point of Sale in PHP. This program mimics how the actual POS work. The functions include:

  • Adding, modification, deletion, and display of products
  • Record the sale of each item in the chart
  • Display description and price of each item sold in the chart
  • Void items in the chart
  • Display description and price of each item sold in the chart
  • Finalize the cart by printing a receipt
  • Automated reduction of items sold in the products
  • Automated warning message if the item is low in quantity
  • View all sales as table by consolidating all receipts
